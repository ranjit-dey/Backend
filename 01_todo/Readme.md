# Todo

## Add Tasks

```bash
node todo.js add "buy milk"
node todo.js add "go to the market"
node todo.js add "pickup the phone"
```

## List all tasks

```bash
node todo.js list
```

## remove single task

```bash
# remove 2nd task (go to the market)
node todo.js remove 2
```

## remove multiple task

```bash
# remove 1st and 3rd task (go to the market)
node todo.js remove 1 3
```
